# גן־ריפורט · Gan Report

אפליקציה לניהול כספי גני ילדים — קצבת עירייה ותשלומי הורים, פעימות הכנסה, קבלות עם OCR, וייצוא אקסל.

## הרצה מקומית
```
npm install
npm run dev
```

## פריסה
מחובר ל‑Netlify. כל push ל‑main נפרס אוטומטית.

### משתני סביבה נדרשים (ב‑Netlify)
- `VITE_ANTHROPIC_API_KEY` — מפתח Anthropic ל‑OCR של קבלות (משמש את הפונקציה בצד השרת)
